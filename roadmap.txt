D-Flow Logic Simulator - Development Roadmap
=============================================

Status: PRE-RELEASE (v0.9.1)
Last updated: 2026-05-10

Completed Milestones
--------------------

[x] Alpha-1: Project Skeleton
    - File structure, index.html, CSS/JS module shells
    - Static UI layout: header, sidebar, canvas, toolbar, footer
    - Design tokens in theme.css, EventBus, IdGenerator

[x] Alpha-2: Professional Styling
    - Dark theme with gradients, shadows, transitions
    - CSS Grid layout with resizable sidebar and splitter
    - Collapsible right panel via PanelManager

[x] Beta-1: Core Engine & Wiring
    - Component base class, ComponentFactory
    - AND, OR, NOT gates + Toggle Switch + LED
    - Simulation engine (queue, step, run, stop)
    - Canvas drag-drop, click-to-wire, Manhattan routing
    - Component dragging with wire follow

[x] Beta-2: Advanced Components & Features
    - All standard gates (NAND, NOR, XOR, XNOR, Buffer, Tri-State)
    - Flip-flops (SR, D, JK, T), SR Latch
    - Clock, DIP Switch (8-bit), 7-Segment Display, Logic Probe
    - Truth table generation, test bench
    - Context menu, property editor, undo/redo

[x] Beta-3: Data, Persistence & Canvas
    - JSON export/import (Serializer.js)
    - Save/Load to localStorage with auto-restore
    - Infinite canvas with zoom/pan
    - Multi-selection (Shift+click, rectangle)
    - Touch support for sidebar drag and canvas

[x] Pre-Release (v0.9): Polish & Bug Fixes
    - Manhattan router with channel-based fan-out avoidance
    - Wire crossing detection with ANSI bridge arcs / IEC junction dots
    - Manual wire editing (control point drag, add/remove)
    - Signal propagation animations
    - Junction dot cleanup fix (ghost gray dots eliminated)
    - Connector dot redesign (8px visual + 24px invisible touch target)
    - Mobile UI/UX improvements (touch targets, spacing, FAB delete button)
    - Three themes: Dark, Light, High Contrast
    - Netlist panel, help overlay, alignment indicators
    - Half Adder, Full Adder, Multiplexer, Shift Register
    - LED Array, High/Low Constant components

[x] v0.9.1 Improvements
    - Enhanced oscillation detection with targeted error marking
    - Step counter and simulation statistics (footer display)
    - Zoom percentage display in footer
    - Circuit minimap with click-to-navigate
    - Example circuits (Half Adder, Full Adder, SR Latch, D Latch, MUX, Counter)
    - Circuit validation (floating inputs, unreachable components)
    - Clear All button with confirmation
    - Collapsible sidebar component groups with icons
    - Component tooltips in sidebar
    - Gate type accent bars (color-coded left border)
    - Component hover highlight effect
    - Wire hover glow effect
    - Wire preview pulse animation
    - Smart Manhattan routing (horizontal alignment, compact C-shape)
    - Refactored zoom with _applyZoom() helper
    - Extended keyboard shortcuts (Ctrl+A, Ctrl+/-, Ctrl+0, Home, F5)
    - Updated help overlay with new shortcuts

Remaining for v1.0 Release
---------------------------

[ ] Performance Optimization
    - Batch SVG updates during high-frequency operations
    - Virtual rendering: only render visible components/wires in viewport
    - Debounce wire color updates during rapid simulation
    - Profile and optimize Router for large circuits (100+ components)

[ ] Accessibility Audit
    - Full ARIA labels on all interactive elements
    - Keyboard-navigable component palette
    - Screen reader announcements for simulation state changes
    - Focus management for modals and panels

[ ] Comprehensive Testing
    - Unit tests for Engine logic (all gate types, flip-flops)
    - Integration tests for wiring and routing
    - Cross-browser testing (Chrome, Firefox, Edge, Safari)
    - Mobile device testing (iOS Safari, Android Chrome)

[ ] Documentation & Examples
    - In-app tutorial / guided walkthrough for new users
    - API documentation for extending components

[ ] Final Polish
    - Consistent error handling and user feedback
    - Wire label / net name display
    - Smooth zoom animation

Future Considerations (v2.0+)
------------------------------

[ ] Timing diagram: visual waveform display for sequential analysis
[ ] Bus routing: multi-bit wire bundles with automatic fan-out
[ ] Collaboration: real-time multiplayer circuit editing via WebSocket
[ ] SPICE export: generate netlist for external simulation tools
[ ] Plugin system: user-defined components with JavaScript API
[ ] FPGA synthesis: generate Verilog/VHDL from circuit
