Logic Gate Simulator - Development Worklog
============================================

[v0.9-pre] - 2026-05-09
Current State: Pre-Release
    Full-featured digital logic circuit simulator with 30+ component types,
    Manhattan wire routing, manual wire editing, multi-selection, undo/redo,
    truth table generation, test bench, netlist viewer, JSON save/load,
    responsive layout, and three color themes.

    Major subsystems:
    - Engine: event-driven propagation, clock sync, infinite loop detection
    - Router: Manhattan (channel-based), Direct, and Manual routing modes
    - Canvas: infinite zoom/pan, SVG wire layer, component drag, alignment guides
    - Wiring: click-to-wire, live preview, auto-reroute on drop, junction detection
    - Wire Edit: manual control point drag, add/remove points, orthogonal constraints
    - Selection: Shift+click, rectangle select, bulk delete, floating mobile button
    - Undo/Redo: full command pattern for add, delete, connect, disconnect, move, property
    - Persistence: JSON serialize/deserialize, localStorage auto-restore
    - Analysis: truth table, test bench, netlist panel
    - Themes: Dark, Light, High Contrast with CSS custom properties
    - Responsive: desktop/tablet/phone breakpoints with overlay sidebar

    Component library:
    Gates:    AND, OR, NOT, NAND, NOR, XOR, XNOR, Buffer, Tri-State Buffer
    FFs:      SR, D, JK, T (edge-triggered), SR Latch
    Chips:    Half Adder, Full Adder, Multiplexer (2:1)
    Sequential: 4-bit Shift Register
    Inputs:   Toggle Switch, DIP Switch (8-bit), Clock, High Constant, Low Constant
    Outputs:  Light Bulb (LED), 7-Segment Display, Logic Probe, LED Array

    Bug fixes applied:
    - Ghost gray dots on canvas after moving components: fixed by clearing
      intermediate junction dots before re-detecting T-connections
    - Connector dots too large on mobile: reduced visual size to 8px with
      invisible 24px pseudo-element hit area for touch-friendliness
    - Mobile UI cramped: improved touch target sizes, spacing, toolbar layout,
      and changed delete button to compact circular FAB style

[v0.8-beta3] - 2026-05-07
    - JSON export/import (Serializer.js) for full circuit state
    - Save/Load buttons to header (localStorage), Export/Import file buttons
    - Sidebar drag-and-drop works on both mouse and touch
    - Canvas infinite size with zoom (wheel/pinch) and pan (middle-mouse/Ctrl+drag)
    - Multi-selection: Shift+click toggle, Shift+drag rectangle select
    - Delete key removes selected components
    - Grid scales with zoom, components clamped to visible viewport

    Critical bug fixes:
    - Flip-flops now set nQ output to true on construction
    - Engine.connect() removes existing wire on overwrite, preventing orphans
    - Re-entrant guard on _processQueue() prevents stack overflow
    - Manhattan backward routing when target is left of source
    - Component border highlights when ANY output is true
    - Engine.addComponent() no longer double-wraps computeOutput() on undo/re-import
    - Component.reset() properly clears sequential state
    - Serializer.importState() clears canvas DOM before importing
